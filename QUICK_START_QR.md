## Troubleshooting

### QR Won't Scan?

1. Try manual code entry (6 digits shown below QR)
2. Refresh QR code (new code generates)
3. Check phone camera focus
4. Increase screen brightness
5. Use different QR scanner app

### Phone Disconnected?

1. Re-scan QR code
2. Or use manual code entry
3. Session remains active
4. Instant reconnection

### Code Expired?

1. Click "Refresh Code" button
2. New QR + code generates instantly
3. Scan new code
4. Transfer completes
