## ‚ö° Panic Button (Boss Key)

**MOST IMPORTANT FEATURE** - Instantly hide the app if interviewer gets suspicious!

### How It Works:
- Press your configured panic key (default: **ESC**)
- App instantly hides or shows decoy screen
- Can also triple-click top-right corner of screen

### Panic Key Options:
- `ESC` - Simple and natural (recommended)
- `Alt+B` - Classic "Boss Key"
- `Ctrl+Shift+H` - Hidden combination
- `F1` - Natural (like opening help)
- `Ctrl+Shift+Space` - Hard to trigger accidentally

### Panic Actions:

#### 1. **Show Decoy Screen** (Recommended)
- Instantly switches to a fake screen
- Available decoys:
  - **Google Search**: "interview preparation best practices"
  - **Stack Overflow**: Technical programming question
  - **GitHub**: Code repository
  - **Technical Documentation**: React/programming docs
  - **Blank Screen**: Plain white page

#### 2. **Minimize Window**
- Minimizes the app instantly
- Can quickly bring back up

#### 3. **Blank Screen**
- Shows completely white screen
- Looks like page is loading

#### 4. **Close App**
- Closes the application
- Most extreme option

### Exiting Decoy Mode:
- Press `ESC` again
- Triple-click top-right corner
- Click hidden X button (appears on hover)

## üîß Troubleshooting Stealth Features

### Panic Button Not Working
- Check keyboard focus is on app
- Try different panic key
- Test in settings page first
- Browser might block some keys

### Floating Widget Not Showing
- Check if PiP is enabled in browser
- Some browsers don't support PiP
- Try different browser (Chrome works best)
- Check browser permissions

### Screen Recording Detection False Positives
- Some browsers trigger it incorrectly
- Disable if too many false alerts
- Works best on Chrome/Edge
- May not work on all systems

### Decoy Screen Not Loading
- Check internet connection (for external decoys)
- Use "blank" decoy as backup
- Test before interview
- Make sure JavaScript enabled

### Mobile Companion Not Connecting
- Both devices must be on same network
- Or use internet connection
- Check firewall settings
- Regenerate code if expired
